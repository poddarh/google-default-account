# Google Default Account

Chrome extension for users with multiple Google accounts. Set a default account for each Google service so you're automatically redirected to the right one.

Supported services: Google Meet, Gmail, Google Drive, Google Photos, YouTube, Google Chat, Google Maps.

## Install

1. Go to `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked** and select this project folder

No build step required.
